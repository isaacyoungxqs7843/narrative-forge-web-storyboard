## Overview

Narrative Forge helps you build story sequences directly in the browser, giving you tools to develop plots, scenes, and visual timing without leaving a web-based workflow. It is a practical fit for writers, designers, and creative teams that want to move from an early premise to a structured narrative environment.

The app blends AI-assisted story creation with live preview capabilities, making it easier to test alternate story paths, refine character relationships, and see how a scene would read in a comic-like format. Built with React and TypeScript, it is aimed at projects that need both narrative planning and presentable visual output.

## Setup

You can get started by cloning the repository or downloading the project files, then opening the web app in a local or hosted environment of your choice.

1. Clone the repository:
   git clone https://github.com/isaacyoungxqs7843/narrative-forge-web-storyboard.git

2. Move into the project folder:
   cd storyboard-studio-ai

3. Install dependencies with your package manager:
   npm install

4. Start the app:
   npm run dev

If you are using the published build, open the download page and launch the latest web version from there.

## How to Use

Begin by starting a new narrative, then add characters, define relationships, and generate a branching sequence from your central idea. After that, you can inspect scenes in the interactive preview, tune the story progression, and render panels whenever you want a more visual presentation.

Typical workflow:

1. Enter a premise or screenplay fragment.
2. Generate a branch of scenes.
3. Review character interactions and consistency checks.
4. Render comic-style panels for selected moments.
5. Export the result in the format you need.

For AI-assisted writing, connect the available model options in the settings and run generation again to compare outputs.
